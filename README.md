# Python Data Utilities

A Flask-based backend project demonstrating Python fundamentals, data processing,
and real-world API design.

## Features
- URL parameter parsing
- Index-based data aggregation
- Ticket allocation system
- Pandas data cleaning and merge
- NumPy matrix analysis

## Tech Stack
- Python
- Flask
- Pandas
- NumPy

## Run the Project
pip install -r requirements.txt
python app.py

## API Endpoints
 POST /parse-url
POST /aggregate
POST /tickets
GET  /pandas-clean
GET  /numpy-analyze
